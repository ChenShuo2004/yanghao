配置说明

详细的配置指南和使用说明

🚀 快速开始

环境准备

在开始使用之前，请确保您的系统满足以下要求：

  • Python 3.8 或更高版本
  • Chrome 浏览器（最新版本）
  • 稳定的网络连接
  • 足够的系统内存（建议8GB以上）

安装步骤：

# 1. 克隆项目
git clone https://github.com/your-repo/youtube-bot.git
cd youtube-bot

# 2. 创建虚拟环境
python -m venv venv

# 3. 激活虚拟环境
# Windows:
.\venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 4. 安装依赖
pip install -r requirements.txt

# 5. 运行程序
python main.py

⚙️ 配置文件说明

config.json 结构

配置文件采用JSON格式，包含以下主要部分：

  • windows：浏览器窗口配置列表
  • settings：全局设置参数
  • behavior：用户行为模拟参数
  • logging：日志记录配置

完整配置示例：

{
  "windows": [
    {
      "browser_id": "Profile 1",
      "window_width": 400,
      "window_height": 600,
      "position_x": 0,
      "position_y": 0
    },
    {
      "browser_id": "Profile 2",
      "window_width": 400,
      "window_height": 600,
      "position_x": 400,
      "position_y": 0
    }
  ],
  "settings": {
    "watch_time_min": 5,
    "watch_time_max": 60,
    "like_probability": 0.1,
    "subscribe_probability": 0.05,
    "scroll_count": 1000,
    "max_videos_per_session": 100
  },
  "behavior": {
    "mouse_speed": 1.0,
    "scroll_pattern": "smooth",
    "pause_probability": 0.15,
    "replay_probability": 0.05,
    "comment_probability": 0.2,
    "user_persona": "casual"
  },
  "logging": {
    "level": "INFO",
    "file": "youtube_bot.log",
    "max_size": "10MB",
    "backup_count": 5
  }
}

📋 参数详解

🖥️ 窗口设置

browser_id

浏览器配置文件ID，用于多账号管理

字符串
window_width/height

浏览器窗口的宽度和高度（像素）

整数 (300-1920)
position_x/y

窗口在屏幕上的位置坐标

整数

⏱️ 时间设置

watch_time_min/max

每个视频的观看时长范围（秒）

整数 (1-600)
scroll_delay

滚动操作之间的延迟时间（毫秒）

整数 (100-5000)
session_duration

单次会话的最大持续时间（分钟）

整数 (10-480)

🎯 交互概率

like_probability

点赞操作的触发概率

浮点数 (0.0-1.0)
subscribe_probability

订阅操作的触发概率

浮点数 (0.0-1.0)
comment_probability

查看评论的触发概率

浮点数 (0.0-1.0)

🎭 行为模拟

user_persona

用户画像类型：casual, active, explorer, focused, random

字符串
mouse_speed

鼠标移动速度倍数

浮点数 (0.5-3.0)
scroll_pattern

滚动模式：smooth, quick, random, lazy

字符串

🔧 高级配置

多ID管理

支持同时管理多个浏览器配置文件，每个ID可以有独立的设置：

  • 不同的观看偏好和行为模式
  • 独立的交互概率设置
  • 分别的日志记录和统计
  • 灵活的窗口布局管理

多ID配置示例：

{
  "windows": [
    {
      "browser_id": "休闲用户",
      "settings": {
        "watch_time_min": 10,
        "watch_time_max": 30,
        "like_probability": 0.05,
        "user_persona": "casual"
      }
    },
    {
      "browser_id": "活跃用户",
      "settings": {
        "watch_time_min": 5,
        "watch_time_max": 45,
        "like_probability": 0.15,
        "subscribe_probability": 0.08,
        "user_persona": "active"
      }
    }
  ]
}

❌ 常见问题

Q: 程序启动后浏览器无法打开？

A: 检查Chrome浏览器是否正确安装，确保chromedriver版本与Chrome版本匹配。

Q: 视频无法自动播放？

A: 确保浏览器允许自动播放，可以在Chrome设置中关闭自动播放限制。

Q: 程序运行一段时间后停止？

A: 检查网络连接，查看日志文件中的错误信息，可能需要调整延迟参数。

Q: 如何查看运行日志？

A: 日志文件默认保存在项目目录下的youtube_bot.log文件中。
