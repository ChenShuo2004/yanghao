安装指南

1

下载文件

选择适合您需求的版本并下载到本地

wget https://releases.github.com/youtube-bot-v2.1.0.zip
2

解压文件

将下载的压缩包解压到您选择的目录

unzip youtube-bot-v2.1.0.zip
cd youtube-bot
3

安装依赖

使用pip安装所需的Python包

pip install -r requirements.txt
4

配置设置

根据需要修改config.json配置文件

cp config.example.json config.json
nano config.json
5

运行程序

启动YouTube自动化机器人

python main.py
